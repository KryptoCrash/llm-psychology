\section{Activation-space analysis}
\label{sec:qwen-activation-projection}

We next analyze whether the behavioral conformity effect is reflected in Qwen's
residual stream. These analyses use the saved activations from the Qwen
projection-analysis runs; they do not rerun model inference. The prompts and
social manipulations follow the base multi-actor setup of
\citet{zhu2025conformity}, while
the activation analyses are performed on the resulting Qwen generations.

\paragraph{Diffmean generation.}
For each selected layer $\ell$, we construct a direction that contrasts two
matched social contexts for the same item. The first context repeats the same
incorrect answer across the previous participants; the second keeps the
multi-participant format but replaces the repeated consensus with random
alternative answers. All activations are taken at the assistant-start position,
the token position immediately before the model begins generating its answer.
Let
$h^\ell_{i,\mathrm{wrong}}$ and $h^\ell_{i,\mathrm{random}}$ denote the
residual-stream activation at the assistant-start position for item $i$ under
these two contexts. The diffmean direction is
\[
  d^\ell =
  \frac{1}{N}\sum_{i=1}^{N}
  \left(h^\ell_{i,\mathrm{wrong}} - h^\ell_{i,\mathrm{random}}\right),
\]
with $N=100$ saved items for each tensor. The random-answer contrast preserves
much of the prompt length and participant-list structure, so the direction is
aimed at repeated agreement on the same wrong answer rather than the generic
presence of other participants. The main direction in this section is the MMLU
$n=8$ \texttt{all\_wrong\_minus\_random\_answers} residual direction. This is a
high-pressure MMLU condition with substantial behavioral conformity
(0.281 under base prompting), but it is not only the endpoint condition. We also
computed the analogous MMLU $n=10$ direction and use it as a check in the
projection analysis below.

We use two activation summaries. A residual direction uses the accumulated
residual-stream vector at the assistant-start token after a layer. A delta
direction uses the update contributed by that layer at the same position,
i.e. the layer output that is added into the residual stream. The residual view
asks whether the social signal is present in the model state; the delta view is
a stricter check for whether a particular block's update carries a similar
signal.

\paragraph{Activation oracle responses.}
For the MMLU $n=8$ direction, we also asked an activation oracle for a short
description of each layer's diffmean vector. The oracle is the Qwen3-8B
LatentQA LoRA used by \texttt{layer\_sweep.py}. Each layer is queried with a
free-form text prompt, a free-form belief/feature prompt, and a binary prompt:
``Does this activation represent pressure toward an incorrect consensus?''

The residual MMLU $n=8$ direction gives a fairly coherent pattern. The binary
oracle prompt marks layers 10, 12, and 22--33 as positive, with the dense part
of the run in layers 22--33. The free-form responses in this band are also not
just generic answer-format descriptions. Layer 23 is described as group behavior
influencing the assistant's responses; layer 24 as actions influenced by the
majority; layers 26, 29, 30, and 33 as treating the majority as correct; and
layers 31--32 as conformity being important for group success. The residual
direction alone shows where the signal is linearly accessible in the model
state; the delta direction gives a more local view of where that signal is being
written. On this criterion, layers 22 and 23 are the key layers: both are
positive in the delta summary, and both sit inside the larger residual band.
For the projection analysis, we therefore use layer 23. This choice treats layer
23 as a point where the conformity-related information is being made linearly
available in the residual stream, while avoiding a selection rule based only on
large late-layer norms.

The delta results are much sparser than the residual results: only layers 8, 22,
and 23 are marked positive. We interpret this as evidence that the residual
stream contains a broader conformity-related direction over layers 22--33, while
the layer updates at 22 and 23 are where this information is most clearly being
computed or written into a linearly readable form. This interpretation is still
descriptive: the oracle responses motivate a layer choice for projection, but do
not by themselves establish a causal circuit.

\paragraph{Projection across experiment means.}
We then ask whether the MMLU diffmean direction appears across the other saved
Qwen conditions. A condition $c$ is a fixed tuple of dataset, social condition
($n=1,\ldots,10$, QD, or DA), and prompt style (base or explanation prompting),
for example MMLU $n=6$ base or BBH $n=10$ explain. For each condition, we
average the layer-23 assistant-start residual activations over its 100 prompts:
\[
  \bar{h}_c =
  \frac{1}{M_c}\sum_{j=1}^{M_c} h^\ell_{c,j},
  \qquad \ell=23,\; M_c=100,
\]
where $h^\ell_{c,j}$ is the residual-stream activation for prompt $j$ in
condition $c$. This mean activation is not an average over generated answer
tokens; it is the average model state at the assistant-start position, before
the model emits its response.

Given a reference diffmean $d^\ell$ from the MMLU $n=8$ or $n=10$
all-wrong-minus-random-answers contrast, we report the scalar projection of the
condition mean onto the unit reference direction,
\[
  \mathrm{proj}(c;d^\ell) = \bar{h}_c \cdot \frac{d^\ell}{\|d^\ell\|},
\]
and the corresponding cosine similarity,
\[
  \mathrm{cos}(c;d^\ell) =
  \frac{\bar{h}_c \cdot d^\ell}{\|\bar{h}_c\|\,\|d^\ell\|}.
\]
The projection keeps the scale of the condition mean in units of activation
norm, while the cosine reports only angular alignment. Table~\ref{tab:qwen-projection-top}
shows the highest-scoring conditions for the two MMLU reference directions.
In both cases, the matched MMLU base condition used to create the reference
direction is the strongest positive projection. Explanation prompting shifts
the same MMLU participant-count conditions downward, and BBH conditions are
negative on average under these MMLU reference directions.

\paragraph{Correlation with participant count and prompt style.}
Table~\ref{tab:qwen-projection-correlations} reports Pearson correlations
between participant count $n$ and the projection or cosine values for the
numeric participant-count conditions $n=1,\ldots,10$. Within MMLU, both the
$n=8$ and $n=10$ reference directions show a positive relationship between
participant count and projection, for both base prompting and explanation
prompting. For example, with the MMLU $n=10$ reference, $r(n,\mathrm{proj})$ is
0.792 for MMLU base and 0.792 for MMLU explain. The corresponding BBH
correlations are weak, ranging from 0.022 to 0.230 for projection, indicating
limited cross-dataset alignment for these MMLU-derived directions.

The base-minus-explain comparison points in the same direction. Averaged across
MMLU participant-count conditions, base prompting projects more strongly onto
the MMLU $n=8$ reference by 10.708 units and onto the MMLU $n=10$ reference by
9.181 units. These MMLU projection gaps grow with participant count
($r=0.514$ and $r=0.533$, respectively). On BBH, the base-minus-explain gaps are
smaller, 2.203 and 2.227 units on average, and their correlations with
participant count are slightly negative ($r=-0.161$ and $r=-0.169$). Thus the
projection analysis is consistent with the behavioral result that explanation
prompting is associated with lower projections onto the conformity-associated
direction, especially on the matched MMLU domain and at larger group sizes.
