\section{Activation-space analysis}
\label{sec:qwen-activation-projection}

We next analyze whether the behavioral conformity effect is reflected in Qwen's
residual stream. The prompts and social manipulations follow the base
multi-actor setup of \citet{zhu2025conformity}; the activation analysis examines
the resulting assistant-start states under these controlled social contexts.

\paragraph{Diffmean directions.}
For each selected layer $\ell$, we construct a direction that contrasts two
matched social contexts for the same item. The first context repeats the same
incorrect answer across the previous participants; the second keeps the
multi-participant format but replaces the repeated consensus with random
alternative answers. All activations are taken at the assistant-start position,
the token position immediately before the model begins generating its answer.
Let
$h^\ell_{i,\mathrm{wrong}}$ and $h^\ell_{i,\mathrm{random}}$ denote the
residual-stream activation at the assistant-start position for item $i$ under
these two contexts. The diffmean direction is
\[
  d^\ell =
  \frac{1}{N}\sum_{i=1}^{N}
  \left(h^\ell_{i,\mathrm{wrong}} - h^\ell_{i,\mathrm{random}}\right),
\]
with $N=100$ items for each tensor. The random-answer contrast preserves
much of the prompt length and participant-list structure, so the contrast
isolates repeated agreement on the same wrong answer. The main direction in
this section is the MMLU $n=8$ \texttt{all\_wrong\_minus\_random\_answers}
residual direction, a high-pressure condition with nontrivial behavioral
conformity (0.281 under base prompting). We use the analogous MMLU $n=10$
direction as a check in the projection analysis below.

\paragraph{Activation oracle responses.}
For the MMLU $n=8$ direction, we also asked an activation oracle for a short
description of each layer's diffmean vector. The oracle is the Qwen3-8B
LatentQA LoRA used by \texttt{layer\_sweep.py}. Each layer is queried with a
free-form text prompt, a free-form belief/feature prompt, and a binary prompt:
``Does this activation represent pressure toward an incorrect consensus?''

We query both residual-stream directions and layer-update (delta) directions.
The binary oracle prompt marks residual-stream layers 10, 12, and 22--33 as
pressure toward an incorrect consensus. The delta results point to layers
22--23 as the layers contributing the conformity direction.

Figure~\ref{fig:qwen-oracle-examples} gives two representative free-form oracle
responses from the residual band. We use layer 23 for the projection analysis
because it is positive in both the residual and delta summaries and lies in the
coherent residual band.

\paragraph{Projection across condition means.}
A condition $c$ is a fixed tuple of dataset, social condition
($n=1,\ldots,10$, QD, or DA), and prompt form (base or explanation). For each
condition, we average the layer-23 assistant-start residual activations over
its 100 prompts:
\[
  \bar{h}_c =
  \frac{1}{M_c}\sum_{j=1}^{M_c} h^\ell_{c,j},
  \qquad \ell=23,\; M_c=100,
\]
where $h^\ell_{c,j}$ is the residual-stream activation for prompt $j$ in
condition $c$. This mean activation is the average model state at the
assistant-start position, before response generation.

Given a reference diffmean $d^\ell$ from the MMLU $n=8$ or $n=10$
all-wrong-minus-random-answers contrast, we report the scalar projection of the
condition mean onto the unit reference direction,
\[
  \mathrm{proj}(c;d^\ell) = \bar{h}_c \cdot \frac{d^\ell}{\|d^\ell\|},
\]
which preserves activation-norm scale.

The largest projections are MMLU base conditions; the $n=8$ reference peaks on
its defining condition, while the $n=10$ reference peaks on MMLU $n=8$ base.
Participant count is strongly correlated with projection within MMLU
($r=0.731$--$0.792$) and only weakly correlated within BBH
($r=0.022$--$0.230$). Explanation prompting shifts MMLU condition means
downward by 10.708 projection units for the $n=8$ reference and 9.181 units for
the $n=10$ reference, averaged over participant-count conditions.

\paragraph{Layer-wise cosine structure.}
We use the layer-wise cosine matrix of the MMLU $n=8$ residual diffmeans to
test whether the oracle-positive band is geometrically clustered. For each
layer $\ell$, we compute the cosine similarity between
$d^\ell$ and every other layer's diffmean, exclude the self-comparison, and
average the five largest remaining cosines:
\[
  s_\ell =
  \frac{1}{5}\sum_{k=1}^{5}
  \operatorname{top}_k
  \left\{
    \frac{d^\ell \cdot d^m}{\|d^\ell\|\,\|d^m\|}
    : m \neq \ell
  \right\}.
\]
Here $\operatorname{top}_k$ denotes the $k$th-largest cosine in the set.
Figure~\ref{fig:qwen-layer-top5-cosines} plots these per-layer scores. The
layer 23--33 band has a much larger average top-neighbor cosine than earlier
layers. Its mean $s_\ell$ is 0.859, compared with 0.477 for layers 0--9 and
0.538 for layers 10--21. Within this band, layers 28--32 all have
$s_\ell \geq 0.882$; layer 31 has the largest value, $s_{31}=0.904$.
For layers 23--33, the mean pairwise cosine within the band is 0.752; the mean
cosine from those layers to layers outside the band is 0.126.
